# <mars-divider>
Web component that allows us to show an accessible and beautiful divider so that users can interact

## Installation
```bash
 npm i @web-inmars/divider --save
```

## Usage
```html
<script type="module">
  import '@web-inmars/divider/mars-divider.js';
</script>

<mars-divider>
    Click me!
</mars-divider>
```
