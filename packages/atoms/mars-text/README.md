# <mars-text>
Web component that allows us to show an accessible and beautiful text so that users can interact

## Installation
```bash
 npm i @web-inmars/text --save
```

## Usage
```html
<script type="module">
  import '@web-inmars/text/mars-text.js';
</script>

<mars-text>
    Click me!
</mars-text>
```
